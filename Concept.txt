

문제: 매년 상장회사 이사회멤버가 변경되고 있다. 누가 선임되는지, 현재 누가 이사인지, 누가 해임되는지 알수가 없다.
목표: DART 사업보고서를 통해서, 누가 선임되고 해임되는지를 자동으로 파악하여, 일목요연하게 보여주는 것

1. 한국 KOSPI 주식시장 시총 상위 10개 기업을 대상으로 한다.
2. 시총 상위 10개 기업의 기업코드를 확보한다.
3. DART 사이트에서, 해당 기업코드 사용하여 최근 10년치 사업보고서 확보한다.
4. 각 사업보고서 내에서 이사선임에 해당되는 내역을 발췌한다.
5. 회사별 연도별 이사회 구성원 현황을 정리한다.
6. 엑셀형식으로 정리해준다.


dart-board-tracker/
├── .env                # DART API KEY 등 환경변수 관리 (보안)
├── .gitignore          # .env 및 엑셀 결과 파일 제외 설정
├── main.py             # 전체 프로세스 제어 (Entry Point)
├── config.py           # 대상 기업 리스트 및 설정값 관리
├── requirements.txt    # 설치 필요한 라이브러리 목록
│
├── src/                # 핵심 로직 폴더
│   ├── __init__.py
│   ├── collector.py    # OpenDartReader를 이용한 보고서 수집
│   ├── parser.py       # BeautifulSoup 기반 HTML 표 추출 및 정제
│   └── analyzer.py     # 연도별 데이터 비교 및 변동 내역 분석
│
└── output/             # 최종 생성된 엑셀 파일 저장 폴더
    └── board_analysis_results.xlsx
